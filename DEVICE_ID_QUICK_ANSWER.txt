═══════════════════════════════════════════════════════════════

❓ ВОПРОС:
"Идентификаторы устройства или другие идентификаторы"
(Device identifiers or other identifiers)

Это включает:
• IMEI-код
• MAC-адрес
• Рекламный идентификатор
• Firebase Instance ID

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ ОТВЕТ: ❌ НЕТ (не отмечать)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ПОЧЕМУ "НЕТ":

✅ Приложение НЕ собирает IMEI
✅ Приложение НЕ собирает MAC-адрес
✅ Приложение НЕ собирает рекламный ID
✅ Приложение НЕ использует Firebase
✅ Нет явного сбора идентификаторов устройства

Что использует:
• Supabase использует User ID (UUID) - это НЕ device ID
• Email адрес - это НЕ device ID
• Session tokens - это НЕ device ID
