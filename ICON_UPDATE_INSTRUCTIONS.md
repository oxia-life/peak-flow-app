# Инструкция по обновлению иконки приложения

## Шаги для обновления иконки

### 1. Подготовка файлов иконок

Вам нужно создать следующие файлы изображений:

- **icon.png** - основная иконка (1024x1024 px)
- **adaptive-icon.png** - адаптивная иконка для Android (1024x1024 px)
- **splash.png** - экран загрузки (1284x2778 px для iPhone 13 Pro Max)
- **favicon.png** - иконка для веб-версии (48x48 px)

### 2. Размещение файлов

Сохраните подготовленные изображения в папку:
```
/Users/monastyrskaya/Documents/peak-flow-diary/assets/
```

Замените существующие файлы:
- `assets/icon.png`
- `assets/adaptive-icon.png`
- `assets/splash.png`
- `assets/favicon.png`

### 3. Формат иконок

**Для icon.png:**
- Размер: 1024x1024 пикселей
- Формат: PNG с прозрачностью
- Используйте изображение с белым символом ветра на градиентном сине-голубом фоне

**Для adaptive-icon.png (Android):**
- Размер: 1024x1024 пикселей
- Формат: PNG
- Учитывайте безопасную зону (central 66% of the image)

**Для splash.png:**
- Размер: 1284x2778 пикселей (или любой другой стандартный размер)
- Можно использовать тот же градиент и символ по центру

**Для favicon.png:**
- Размер: 48x48 пикселей
- Упрощенная версия основной иконки

### 4. Очистка кэша после обновления

После замены файлов запустите:
```bash
cd /Users/monastyrskaya/Documents/peak-flow-diary
npx expo start --clear
```

### 5. Настройки в app.json

Текущие настройки (уже правильно настроены):
```json
{
  "expo": {
    "icon": "./assets/icon.png",
    "splash": {
      "image": "./assets/splash.png",
      "backgroundColor": "#ffffff"
    },
    "android": {
      "adaptiveIcon": {
        "foregroundImage": "./assets/adaptive-icon.png",
        "backgroundColor": "#ffffff"
      }
    },
    "web": {
      "favicon": "./assets/favicon.png"
    }
  }
}
```

### 6. Быстрое создание иконок онлайн

Вы можете использовать онлайн-инструменты для создания всех нужных размеров:
- https://appicon.co/ - генерирует все размеры иконок
- https://hotpot.ai/icon-resizer - ресайзер иконок
- https://makeappicon.com/ - создание иконок для всех платформ

Загрузите ваше изображение (белый символ ветра на градиенте) и скачайте готовый набор иконок.

### 7. Проверка

После обновления иконок:
1. Перезапустите Metro Bundler с очисткой кэша
2. Проверьте иконку в Expo Go
3. При сборке приложения иконки будут применены автоматически
