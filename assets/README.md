# Assets

Эта папка содержит ресурсы приложения (иконки, изображения).

## Необходимые файлы

Для работы приложения нужно добавить следующие файлы:

- `icon.png` - иконка приложения (1024x1024px)
- `splash.png` - экран загрузки (1242x2436px)
- `adaptive-icon.png` - адаптивная иконка для Android (1024x1024px)
- `favicon.png` - favicon для web версии (48x48px)

## Временное решение

Пока файлы не добавлены, Expo будет использовать стандартные иконки.

Вы можете:
1. Создать свои иконки
2. Использовать генератор: https://www.appicon.co/
3. Или оставить стандартные от Expo (для разработки)

## Рекомендации

- Формат: PNG с прозрачностью
- Иконка должна быть простой и узнаваемой
- Для медицинского приложения рекомендуются чистые цвета (белый, синий, зелёный)
- Избегайте мелких деталей - они будут плохо видны на маленьких иконках
